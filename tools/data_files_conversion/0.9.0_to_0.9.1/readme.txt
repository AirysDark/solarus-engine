The formats of dialog files and of the language list file has changed in
Solarus 0.9.1.

To upgrade your quest data files from Solarus 0.9.0 to Solarus 0.9.1, first
backup all your Solarus 0.9.0 quest data files. Then, type:
lua update_quest.lua path/to/your_quest
(you need the Lua 5.1 interpreter).
